# Location Management Module

Este módulo manejará la gestión de ubicaciones (sedes, edificios, pilas).

## Estructura Propuesta

```
location-management/
├── components/
│   ├── location-list/
│   ├── sede-management/
│   ├── edificio-management/
│   └── pila-management/
├── services/
│   └── location.service.ts
├── models/
│   └── location.interface.ts
└── location-management.routes.ts
```
